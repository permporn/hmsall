ทำ Autocomplete เพื่อแสดงข้อมูลในฐานข้อมูลตามที่คนดูกรอก ให้กับ Textbox

*วิธีการติดตั้ง*

1. ให้ ไปที่ Control Panel ของเว็บไซต์ของคุณ สร้างฐานข้อมูลชื่อ php_ex (จะเป็นชื่ออื่นก็ได้นะ)
2. ใช้โปรแกรม Notepad (อยู่ที่ Start -> All Programs -> Accessories -> Notepad) เปิด File ชื่อ dbconfig.php ครับ
3. ไปแก้ไข Code ภายในนี้ครับ (เป็นค่าที่เกี่ยวกับการเชื่อมต่อฐานข้อมูล ตั้งให้สอดคล้องกับที่สร้างไว้ในข้อ 1 นะ) แก้เสร็จแล้วก็ Save ครับ (ไปที่ File --> Save นะ)
4. ทีนี้ให้คุณ UPLOAD ไฟล์ทั้งหมดขึ้นไปบน Host เลยครับ
5. ให้ใช้ BROWSE เปิดหน้าชื่อ setup.php เพื่อทำการติดตั้ง Autocomplete จะพบหน้าแสดงผลดังภาพด้านล่าง (ใส่ URL ของไฟล์นี้ลงใน ADDRESS BAR ตัวอย่างเช่น http://www.sitename.com/autocomplete/setup.php )
6. เป็นอันเสร็จสิ้นขั้นตอนการติดตั้งแล้วครับ
7. ใครมีข้อสงสัยอะไรก็อีเมล์มาถามกันได้นะครับที่ au_cty@hotmail.com ครับ
